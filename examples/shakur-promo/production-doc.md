# Example: Boxing Promo — "The Chosen Wan"

2-minute promo interleaving real footage with anime sequences.
Three-act structure: The Grind → The Ring → The Future.

## Pipeline Used
1. Raw photos/clips uploaded
2. GPT-4o converted key frames to anime (Hajime no Ippo style)
3. Grok Imagine animated the anime keyframes
4. Audio via Suno V5
5. Assembly in CapCut

See the main docs/TESTING.md for quality validation results.
